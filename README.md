# 🧑‍🤝‍🧑 Mini Social App

A minimal social media web application where users can:
- Register and log in
- Upload a profile picture
- Post text-based content to a shared feed
- View posts from all users
- View user profile

> Built with **Node.js**, **Express**, **MongoDB Atlas**, **Mongoose**, **EJS**, **Multer**, and **Tailwind CSS**

---

## 🔧 Features

- 🔐 User Authentication (Sign up, Login, Logout)
- 🖼️ Profile Picture Upload using Multer
- ⚡️ Edit Post
- 💜 Like and Dislike Post
- 📰 Global Feed to View All Posts
- 🧑 View Your Own Profile
- 💾 MongoDB Atlas Integration
- 🎨 Clean UI with Tailwind CSS

---

## 💻 Tech Stack

| Tech             | Usage                         |
|------------------|-------------------------------|
| **Node.js**      | JavaScript runtime environment |
| **Express**      | Web framework for Node.js      |
| **MongoDB Atlas**| Cloud-hosted NoSQL database    |
| **Mongoose**     | MongoDB object modeling        |
| **Multer**       | Middleware for file uploads    |
| **EJS**          | Server-side templating engine  |
| **Tailwind CSS** | Utility-first CSS framework    |
| **dotenv**       | Environment variable management|
| **bcrypt**       | Password hashing               |
| **jsonwebtoken** | User session & auth tokens     |
